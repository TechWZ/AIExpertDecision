# AIExpertDecision
AI Expert Decision System

以下以macOS为例：
## frontend
1. 安装依赖
`npm install`
2. 以开发模式运行
`npm run dev`
3. 编译构建生产版本
`npm build`

## backend
1. 创建虚拟环境
`python3 -m venv .venv`
2. 激活虚拟环境
`source .venv/bin/activate`
4. 安装依赖
`pip install -r requirements.txt` 
6. 执行
`.venv/bin/python3 run.py`
